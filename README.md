## Screenshot to OCR, and Text Isolator

Given an image with text, and an image without. This will isolate the text and pass it through Tesseract using pytesseract.

#### Prerequisites

```
Python 3.6+
Tesseract 3.05 or 4.00+
```
##### Libraries
```
cv2
pytesseract
numpy
Pillow
pyperclip
```
